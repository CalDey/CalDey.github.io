实现CSS水平垂直居中的8种方法

发表于 2020-12-20 分类于 CSS

实现CSS水平垂直居中是一道面试的必考题，看似简单，其实是对CSS基础知识的全面检测。

1.常见的CSS居中方式

常见的CSS水平垂直居中的方式大概有以下两大类：

元素居中定宽高：

  • absolute + 负margin
  • absolute + margin:auto
  • absolute + calc

元素居中不定宽高：

  • absolute + transform
  • lineheight
  • css-table
  • flex
  • grid
