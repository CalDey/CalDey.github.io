JavaScript-内置对象数组

发表于 2021-01-08 更新于 2021-01-09 分类于 JavaScript

阅读全文，了解更多

可以通过Array.prototype查看数组一共有多少种内置方法：

下面我们来一一介绍这些方法

1.Array方法

1.1.Array.from()

从一个类数组或可迭代对象中创建一个新的，浅拷贝数组实例

1.2.Array.isArray()

判断一个某个变量是否是数组对象

1.3.Array.of()

创建一个拥有不定参数的新数组，且不考虑参数的类型

2.Array.prototype方法

所有数组实例都通过原型链从Array.prototype继承属性和方法，这些方法主要分为两种，修改原数组值的方法和不会修改原数组值的方法，下面来分别介绍：

2.1.会修改属性值的方法

2.1.1.Array.prototype.copyWithin()

arr.copyWithin(target,start,end) (start,end可为空)

复制数组中某个区间的数据到指定位置，会修改原数组

点我了解更多

2.1.2.Array.prototype.fill()

用一个固定值填充数组从起始索引到终止索引的值，不包括终止索引，原数组会改变

2.1.3.Array.prototype.pop()

从数组中删除最后一个元素的值，然后返回删除元素的值，会修改原数组

2.1.4.Array.prototype.push()

在数组末尾添加一个或多个元素，并返回新数组的长度，会修改原数组

2.1.5.Array.prototype.reverse()

翻转数组，会修改原数组

2.1.6.Array.prototype.shift()

删除数组第一个元素，并返回这个元素，会修改原数组

2.1.7.Array.prototype.sort()

对数组进行排序，并返回当前数组，会修改原数组

2.1.8.Array.prototype.splice()

替换删除或添加元素来修改数组，会修改原数组

2.1.9.Array.prototype.unshift()

添加一个或多个元素到数组的首部，并返回新数组的长度，会修改原数组

2.2.不会修改原数组的方法

2.2.1.Array.prototype.concat()

返回一个由当前数组和若干个其他数组组成的新数组，不会修改原数组

2.2.2.Array.prototype.includes()

判断一个数组是否含有某一特定值，根据情况返回true或者false，不会修改原数组

2.2.3.Array.prototype.join()

将一个数组的所有元素连接成一个字符串并返回这个字符串，不会修改原数组

2.2.4.Array.prototype.slice()

返回一个新数组，是由begin和end决定的对原数组的浅拷贝，数组中包括begin不包括end，不会修改原数组

2.2.5.Array.prototype.toString()

返回数组的字符串形式，不会修改原数组

2.2.6.Array.prototype.indexOf()

返回一个指定元素在数组中的第一个索引，不存在返回-1，不会修改原数组

2.3.迭代方法

在下面这些遍历方法中，有许多方法需要指定一个回调函数作为参数

2.3.1.Array.prototype.forEach()

对数组内每个元素都执行一次给定的函数

forEach可以改变原数组

2.3.2.Array.prototype.filter()

创建一个新数组，其包含的元素是通过filter提供的函数测试通过后的所有元素
